# Product Overview

Photo Sampler is a CLI tool that extracts frames from video files at a configurable frames-per-second rate.

Key capabilities:
- Extract frames at any FPS rate (including fractional, e.g. 0.5 fps)
- Timestamp-based filenames in `frame_MM_SS_mmm.png` format
- Optional frame resizing with Lanczos interpolation
- Supports MP4, AVI, MOV, MKV, and WebM formats
- Lossless PNG output
